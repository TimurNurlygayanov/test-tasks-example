How To Install Requirements
---------------------------

```bash
pip3 install allpairspy
pip3 install pytest pytest-xdist
```

How To Run
----------

To run simple example:
```bash
python3 generate.py
```

Example of how to use it in PyTest test cases:
```bash
python3 -m pytest -n 10 test_me.py -v
```
